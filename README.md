# 🎉 Simon Game

This is a web-based version of the classic **Simon Game**, where you test your memory skills by repeating an ever-growing sequence of colors and sounds.

## 🚀 Features
- Interactive and responsive design
- Smooth animations and sound effects
- Endless levels — see how far you can go!
- Built with vanilla **HTML**, **CSS**, and **JavaScript**

## 🎯 How to Play
1. Click **Start** to begin.
2. Watch the sequence of colors and listen to the tones.
3. Repeat the sequence in the correct order.
4. Each round adds a new color — try not to miss!
